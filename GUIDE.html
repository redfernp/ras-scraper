Setup & User Guide — how to install and use the app

A
Before You Start — Install These Once

You only do this once. After these are installed you never need to touch them again.

1
Install Python 3.11
Go to python.org/downloads/release/python-3119 and download the Windows installer (64-bit).
Run it — before clicking Install Now, tick the checkbox at the bottom that says "Add Python to PATH".
2
Install Git
Go to git-scm.com/download/win. Under Standalone Installer, click Git for Windows/x64 Setup (the first option — this is correct for almost all Windows laptops).
Run the installer and click Next all the way through — all the default options are fine.
Once Python 3.11 and Git are installed, you will never need to do this again.
B
First-Time Setup — Download the App

You only do this once. It downloads the app and installs everything it needs.

1
Save the file clone_and_setup.bat to your Desktop (your administrator will send this to you).
2
Double-click clone_and_setup.bat.
A black window opens and downloads the app — this takes 1–2 minutes. Text will scroll past — this is normal, just let it run.
3
When the black window says "Setup complete!", press any key to close it.
A folder called ras-scraper will appear in the same place as the setup file.
4
Open the ras-scraper folder. Inside you will see a file called run.bat — this is what you use every day from now on.
You will never need to run clone_and_setup.bat again.
1
Every Day — Launching the App
1
Open the ras-scraper folder and double-click run.bat.
Or use the RAS Tips shortcut on your Desktop if one was set up for you.
2
A black window opens briefly — it checks for updates automatically, then launches Chrome and opens the app in your browser. Do not close the black window while using the app — you can minimise it.
3
The app opens at http://localhost:8501 in your browser. If it doesn't open automatically, type that address into your browser.
Every time you run the app it silently downloads the latest version — you never need to update manually.
2
Using the App — Generating Tips
1
In the app, click Step 1 — Launch Chrome with remote debugging to expand it, then click the 🚀 Launch Chrome button. A Chrome window will open.
2
In that Chrome window, go to racingandsports.com.au. If you see a security check, click through it. Once the site loads normally, come back to the app tab.
3
Choose your mode using the selector in the app:
Default

Today's Tips

Automatically finds all of today's races for your countries. Nothing to fill in — just click Generate Tips.

Future dates

Add Racecard URLs

Paste meeting links manually. Use this to prepare Sunday's tips on a Friday or Saturday.

4
Click ▶ Generate Tips and wait. The app shows progress as it works through each race meeting. This usually takes 2–5 minutes.
5
Results appear on screen with each selection, gap score, and NAP / NB badges. A copy-paste block appears at the bottom for easy sharing, and the tips are automatically saved to a file in the ras-scraper folder.
Using Add Racecard URLs? Paste the full meeting URL from racingandsports.com.au, one per line. Example:

https://www.racingandsports.com.au/form-guide/thoroughbred/ireland/leopardstown/2026-04-06
3
Understanding the Results
🥇
NAP — The race with the biggest points gap between the top two rated horses. The highest confidence selection of the day.
🥈
NB — The race with the second biggest points gap. Second highest confidence.
📊
Gap < 4 pts — When the top two horses are close in points, the 2nd rated horse is selected instead of the 1st.
💾
Tips are automatically saved to tips_YYYY-MM-DD.txt inside the ras-scraper folder each time you run the app.

Troubleshooting
The app doesn't open in my browser after double-clicking run.bat
Wait 10 seconds, then open your browser and go to http://localhost:8501 manually.
Chrome not found — the app shows a red error
Click the Launch Chrome button inside the app first, then visit racingandsports.com.au and pass the security check before clicking Generate Tips.
I see a Cloudflare / security check on racingandsports.com.au
Complete the check in the Chrome window that opened. Once the site loads normally, go back to the app tab and click Generate Tips.
No meetings found for my countries today
There may genuinely be no racing today for those countries. Try Add Racecard URLs mode and paste a specific meeting link to check.
A race shows a dash (—) instead of a horse name
No tips data was found for that race — it may not have a form guide yet or the page timed out. All other races are unaffected.
The black window closed immediately when I double-clicked run.bat
Something went wrong at startup. Contact your administrator and describe what you saw.
Remember: Never close the black command window while using the app — it keeps everything running. Minimise it instead.
