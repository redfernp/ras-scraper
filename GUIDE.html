User Guide — Getting started and daily use

✓
Before You Start — One-Time Requirements

You only need to install these two things once. If you have already done this, skip to Section 1.

A
Install Python
Go to python.org/downloads and download the latest version.
Important: During installation, tick the checkbox that says "Add Python to PATH" before clicking Install Now.
B
Install Git
Go to git-scm.com/download/win and download the installer.
Run it and click Next through all the steps — the defaults are fine.
Once Python and Git are installed, you will never need to do this again.
1
First-Time Setup — Download the App

You only do this once. It downloads the app to your computer.

1
Save the file clone_and_setup.bat to your Desktop (your administrator will send this to you).
2
Double-click clone_and_setup.bat.
A black window will open and download the app — this takes about a minute.
3
When it says "Setup complete!" press any key to close the window.
You will see a new folder called ras-scraper appear on your Desktop (or wherever you saved the setup file).
You will not need to run clone_and_setup.bat again after this.
2
Every Day — Running the App
1
Open the ras-scraper folder and double-click run.bat.
(Or use the desktop shortcut if one has been created for you.)
2
A black window will appear briefly — it checks for updates and then opens Chrome and the app in your browser automatically.
3
If you see a security check on racingandsports.com.au — complete it in Chrome. Then come back to the app tab.
4
In the app, paste the race meeting URLs — one per line — into the box.
5
Click ▶ Generate Tips and wait. Results appear below with your NAP and NB selections highlighted.
6
Use the Copy-paste version at the bottom to grab the tips as plain text.
The app automatically downloads the latest version every time you run it — you never need to update it manually.
3
Understanding the Results
🥇
NAP — The race with the biggest points gap between the top two rated horses. Highest confidence selection.
🥈
NB — The race with the second biggest points gap. Second highest confidence selection.
📊
Gap < 4 pts — When the gap between the top two is small, the 2nd rated horse is selected instead of the 1st.

Troubleshooting
The app says "Chrome not found on port 9222"
Close the black window, then double-click run.bat again. Make sure Chrome fully opens before clicking Generate Tips.
The page just shows a Cloudflare / security check
Complete the check in the Chrome window that opened. Once you pass it, go back to the app tab and click Generate Tips again.
The black window says "git is not recognised"
Git is not installed. Follow step B in the "Before You Start" section, then try again.
The app opened but shows no results / error on a race
The website may have changed layout for that race. Try again in a few minutes, or contact your administrator.
The black window closed immediately with no message
Right-click run.bat, choose Open with > Notepad to see the file, or contact your administrator.
Don't close the black window while you are using the app — it keeps the app running. You can minimise it.
